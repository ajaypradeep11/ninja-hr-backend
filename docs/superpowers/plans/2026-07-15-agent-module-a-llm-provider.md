**Goal:** A vendor-neutral `LlmProvider` contract with a resilient Gemini implementation, a thin Anthropic wrapper, an `AiModule` exposing the `LLM_PROVIDER_CHAT` DI token, plus the consolidated Prisma schema (all six new agent models) that Modules B–E build on.

## Global Constraints

- Interface names are frozen by the design spec: `LlmMessage`, `LlmRequest`, `LlmResult`, `LlmProvider`, `LlmClassifier`, DI token `LLM_PROVIDER_CHAT`. Other modules are being planned in parallel against these exact names — do not rename anything.
- Gemini models are env-overridable with these exact defaults: `GEMINI_MODEL` → `gemini-3.5-flash`, `GEMINI_CLASSIFIER_MODEL` → `gemini-2.5-flash-lite`, `GEMINI_EMBED_MODEL` → `gemini-embedding-2`.
- API keys (`GEMINI_API_KEY`, `ANTHROPIC_API_KEY`) are read **at call time**, never captured at construct/import time (keys rotated after boot must be honored).
- Bounded timeout 60,000 ms, exactly 1 retry (SDK `retryOptions.attempts: 2` = original + 1 retry), errors logged and degraded — never crash the caller.
- `safety: 'strict'` maps to `HarmBlockThreshold.BLOCK_LOW_AND_ABOVE` on all four Gemini-API-supported harm categories (harassment, hate speech, sexually explicit, dangerous content).
- `promptFeedback.blockReason` maps to `LlmResult.blocked`.
- **No existing service is rewritten in this project** — `copilot.service.ts`, `jd-generator.service.ts`, etc. stay untouched.
- `assertProductionConfig()` in `src/main.ts` is **not** extended (a missing Gemini key degrades gracefully; it is not an auth bypass).
- Tests are colocated `*.spec.ts` next to the code under `src/` (jest `rootDir: src`); SDKs are mocked with `jest.mock`.
- Migrations are dated/sequential under `prisma/migrations/`; never hand-edit an applied migration. `npm run prisma:migrate` runs `migrate deploy` — for creating a new migration use `prisma migrate dev` (see Task 6).
- No streaming anywhere (output guardrails need the complete text).

```typescript
// src/platform/ai/llm-provider.ts
// Vendor-neutral LLM contract. Interface shapes are copied verbatim from the
// approved design (docs/superpowers/specs/2026-07-15-hr-ai-agent-gemini-guardrails-design.md)
// and are frozen — Modules B–E are built against these exact names.

export interface LlmMessage {
  role: 'user' | 'assistant';
  content: string;
}

export interface LlmRequest {
  system: string;
  messages: LlmMessage[];
  maxTokens: number;
  temperature?: number;
  /** Provider-native safety level; 'strict' maps to BLOCK_LOW_AND_ABOVE on Gemini. */
  safety?: 'strict' | 'default';
  /** Optional inline document (handbook PDF extraction). */
  document?: { base64: string; mimeType: string };
}

export interface LlmResult {
  text: string;
  /** Set when the provider itself refused/blocked (e.g. promptFeedback.blockReason). */
  blocked?: { reason: string };
}

export interface LlmProvider {
  complete(req: LlmRequest): Promise<LlmResult>;
  embed(texts: string[]): Promise<number[][]>;
  /** False when no API key is configured — callers use their template fallbacks. */
  isLive(): boolean;
}

/**
 * Narrow guardrails-classifier contract (Module B) so the core LlmProvider
 * interface stays vendor-neutral. GeminiProvider implements it with the
 * GEMINI_CLASSIFIER_MODEL; returns the model's raw text (the guard parses it).
 */
export interface LlmClassifier {
  classify(system: string, text: string): Promise<string>;
}

/** DI token: the chat-grade provider selected by AI_PROVIDER_CHAT (default 'gemini'). */
export const LLM_PROVIDER_CHAT = 'LLM_PROVIDER_CHAT';

/** DI token: the guardrails input classifier (always Gemini — the classifier model is Gemini-only). */
export const LLM_CLASSIFIER = 'LLM_CLASSIFIER';

/**
 * Thrown when a provider cannot serve: no API key configured, timeout after
 * the single retry, or any SDK error. Callers catch this and degrade to their
 * deterministic template fallbacks — the same "degrade, never crash" rule as
 * copilot.service.ts. The provider logs the underlying cause before throwing,
 * so a revoked key or rate limit is never silently indistinguishable from
 * "no key".
 */
export class LlmUnavailableError extends Error {
  constructor(message: string) {
    super(message);
    this.name = 'LlmUnavailableError';
  }
}
```

**Interfaces:**
- Consumes (from Task 1, `src/platform/ai/llm-provider.ts`): `LlmProvider`, `LlmClassifier`, `LlmRequest`, `LlmResult`, `LlmUnavailableError`.
- Produces: `class GeminiProvider implements LlmProvider, LlmClassifier` with `isLive(): boolean` and `complete(req: LlmRequest): Promise<LlmResult>` working; `embed()`/`classify()` exist as rejecting stubs until Task 3 replaces them. Behavior contract for later modules:
  - `isLive()` is true iff `GEMINI_API_KEY` is a non-empty env var, checked at call time.
  - `complete()` throws `LlmUnavailableError` when no key or when the SDK call fails (after the SDK's 1 retry); it never returns partial data on error.
  - A provider-blocked prompt returns `{ text: '', blocked: { reason: '<blockReason>' } }` — it does NOT throw.

**Background for the implementer:**
- `@google/genai` usage: `new GoogleGenAI({ apiKey, httpOptions })` then `await ai.models.generateContent({ model, contents, config })`. `config.systemInstruction` takes the system string; `config.safetySettings` takes `{ category: HarmCategory.X, threshold: HarmBlockThreshold.BLOCK_LOW_AND_ABOVE }[]`; `config.maxOutputTokens` / `config.temperature` map from our request. `httpOptions.timeout` is in ms; `httpOptions.retryOptions.attempts` counts the original request (2 = 1 retry; the SDK defaults to 5 if unset, so it must be set explicitly).
- `contents` is `Content[]` = `{ role: 'user' | 'model', parts: Part[] }[]`. Gemini's assistant role is `'model'`, so `LlmMessage.role === 'assistant'` must be mapped. An inline document is a part: `{ inlineData: { data: <base64>, mimeType } }`.
- The response exposes `response.text` (concatenated text parts, may be `undefined`) and `response.promptFeedback?.blockReason` (set when the prompt itself was blocked).

```typescript
// src/platform/ai/gemini.provider.ts
import { Injectable, Logger } from '@nestjs/common';
import { GoogleGenAI, HarmBlockThreshold, HarmCategory } from '@google/genai';
import type { Content, GenerateContentConfig, Part, SafetySetting } from '@google/genai';
import { LlmUnavailableError } from './llm-provider';
import type { LlmClassifier, LlmProvider, LlmRequest, LlmResult } from './llm-provider';

// Read at call time (not import/construct time) so keys set/rotated after boot
// are honored; presence-only check — the SDK is the authority on validity.
// Same rule as copilot.service.ts.
function liveKey(): string | undefined {
  return process.env.GEMINI_API_KEY || undefined;
}

// Hard ceiling on the API round trip. copilot.service.ts caps at 20s for the
// quick-ask drawer; the design raises this to 60s for long-form drafting.
// attempts: 2 = the original request + exactly 1 retry (SDK default is 5 —
// must be set explicitly).
const API_TIMEOUT_MS = 60_000;
const RETRY_ATTEMPTS = 2;

// 'strict' → strictest blocking tier on the four harm categories the Gemini
// API supports (HARM_CATEGORY_JAILBREAK is Vertex-only; injection detection
// lives in the guardrails input guard, Module B).
const STRICT_SAFETY: SafetySetting[] = [
  { category: HarmCategory.HARM_CATEGORY_HARASSMENT, threshold: HarmBlockThreshold.BLOCK_LOW_AND_ABOVE },
  { category: HarmCategory.HARM_CATEGORY_HATE_SPEECH, threshold: HarmBlockThreshold.BLOCK_LOW_AND_ABOVE },
  { category: HarmCategory.HARM_CATEGORY_SEXUALLY_EXPLICIT, threshold: HarmBlockThreshold.BLOCK_LOW_AND_ABOVE },
  { category: HarmCategory.HARM_CATEGORY_DANGEROUS_CONTENT, threshold: HarmBlockThreshold.BLOCK_LOW_AND_ABOVE },
];

@Injectable()
export class GeminiProvider implements LlmProvider, LlmClassifier {
  private readonly logger = new Logger(GeminiProvider.name);

  isLive(): boolean {
    return Boolean(liveKey());
  }

  async complete(req: LlmRequest): Promise<LlmResult> {
    const ai = this.client();
    const model = process.env.GEMINI_MODEL || 'gemini-3.5-flash';
    const config: GenerateContentConfig = {
      systemInstruction: req.system,
      maxOutputTokens: req.maxTokens,
    };
    if (req.temperature !== undefined) config.temperature = req.temperature;
    if (req.safety === 'strict') config.safetySettings = STRICT_SAFETY;
    try {
      const response = await ai.models.generateContent({ model, contents: this.toContents(req), config });
      const blockReason = response.promptFeedback?.blockReason;
      if (blockReason) return { text: '', blocked: { reason: String(blockReason) } };
      return { text: (response.text ?? '').trim() };
    } catch (err) {
      throw this.unavailable('generateContent', err);
    }
  }

  // Replaced with a real implementation in the next task (embed/classify).
  embed(_texts: string[]): Promise<number[][]> {
    return Promise.reject(new LlmUnavailableError('GeminiProvider.embed not implemented yet'));
  }

  classify(_system: string, _text: string): Promise<string> {
    return Promise.reject(new LlmUnavailableError('GeminiProvider.classify not implemented yet'));
  }

  /** Throws LlmUnavailableError when no key is configured — callers use their fallbacks. */
  private client(): GoogleGenAI {
    const apiKey = liveKey();
    if (!apiKey) throw new LlmUnavailableError('GEMINI_API_KEY is not configured');
    return new GoogleGenAI({
      apiKey,
      httpOptions: { timeout: API_TIMEOUT_MS, retryOptions: { attempts: RETRY_ATTEMPTS } },
    });
  }

  /** Gemini's assistant role is 'model'; an inline document rides as a part on the last user turn. */
  private toContents(req: LlmRequest): Content[] {
    const contents: Content[] = req.messages.map((m) => ({
      role: m.role === 'assistant' ? 'model' : 'user',
      parts: [{ text: m.content }],
    }));
    if (req.document) {
      const docPart: Part = { inlineData: { data: req.document.base64, mimeType: req.document.mimeType } };
      const last = contents[contents.length - 1];
      if (last && last.role === 'user' && last.parts) last.parts.push(docPart);
      else contents.push({ role: 'user', parts: [docPart] });
    }
    return contents;
  }

  /** Degrade, never crash — but leave an operator signal (revoked key / rate limit ≠ "no key"). */
  private unavailable(op: string, err: unknown): LlmUnavailableError {
    const message = err instanceof Error ? err.message : String(err);
    this.logger.error(`Gemini ${op} failed: ${message}`);
    return err instanceof LlmUnavailableError ? err : new LlmUnavailableError(`Gemini ${op} failed: ${message}`);
  }
}
```

```typescript
// src/platform/ai/anthropic.provider.ts
// Thin wrapper over the existing @anthropic-ai/sdk usage pattern
// (copilot.service.ts) so future migrations of Anthropic-backed features are a
// DI swap. No existing service is rewritten in this project.
import { Injectable, Logger } from '@nestjs/common';
import Anthropic from '@anthropic-ai/sdk';
import { LlmUnavailableError } from './llm-provider';
import type { LlmProvider, LlmRequest, LlmResult } from './llm-provider';

// Read at call time (not import time) so keys set/rotated after boot are honored.
function liveKey(): string | undefined {
  return process.env.ANTHROPIC_API_KEY || undefined;
}

const API_TIMEOUT_MS = 60_000;
// Same model as copilot.service.ts — this wrapper introduces no new Anthropic behavior.
const MODEL = 'claude-opus-4-8';

@Injectable()
export class AnthropicProvider implements LlmProvider {
  private readonly logger = new Logger(AnthropicProvider.name);

  isLive(): boolean {
    return Boolean(liveKey());
  }

  // NOTE: req.safety is a no-op here — Anthropic has no safetySettings
  // equivalent; guardrail stages 1 and 3 (Module B) still wrap every call.
  async complete(req: LlmRequest): Promise<LlmResult> {
    const apiKey = liveKey();
    if (!apiKey) throw new LlmUnavailableError('ANTHROPIC_API_KEY is not configured');
    if (req.document) {
      // Inline documents (handbook PDF extraction) are a Gemini-path feature; keep this wrapper thin.
      throw new LlmUnavailableError('AnthropicProvider does not support inline documents');
    }
    try {
      const client = new Anthropic({ apiKey, timeout: API_TIMEOUT_MS, maxRetries: 1 });
      const response = await client.messages.create({
        model: MODEL,
        max_tokens: req.maxTokens,
        ...(req.temperature !== undefined ? { temperature: req.temperature } : {}),
        system: req.system,
        messages: req.messages.map((m) => ({ role: m.role, content: m.content })),
      });
      const text = response.content
        .filter((b): b is Anthropic.TextBlock => b.type === 'text')
        .map((b) => b.text)
        .join('')
        .trim();
      return { text };
    } catch (err) {
      // Degrade, never crash — but leave an operator signal.
      const message = err instanceof Error ? err.message : String(err);
      this.logger.error(`Anthropic API call failed: ${message}`);
      throw new LlmUnavailableError(`Anthropic API call failed: ${message}`);
    }
  }

  embed(): Promise<number[][]> {
    // A plain Error, not LlmUnavailableError: calling this is a programming
    // error (embeddings run on the Gemini provider), not a degradable condition.
    return Promise.reject(new Error('Not implemented: AnthropicProvider.embed — embeddings run on the Gemini provider'));
  }
}
```

```typescript
import { AnthropicProvider } from './anthropic.provider';
import { GeminiProvider } from './gemini.provider';
import { LLM_CLASSIFIER, LLM_PROVIDER_CHAT } from './llm-provider';
```

```typescript
 * time) — switching providers is a deploy-time decision; API keys, by
 * contrast, are read at call time inside each provider.
 * Exported for direct unit testing.
 */
export function resolveChatProvider(gemini: GeminiProvider, anthropic: AnthropicProvider): LlmProvider {
  const choice = (process.env.AI_PROVIDER_CHAT || 'gemini').trim().toLowerCase();

@Module({

    GeminiProvider,

    { provide: LLM_CLASSIFIER, useExisting: GeminiProvider },
  ],
  exports: [LLM_PROVIDER_CHAT, LLM_CLASSIFIER, GeminiProvider],
})
export class AiModule {}
```
